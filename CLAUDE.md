# Claude Daily Finance Routine Instructions

This repository stores automated daily financial news reports generated by Claude Routine.

## Main task

Each routine run must generate one new markdown report for the **previous calendar day** using Australia/Sydney time (AEST/AEDT).

**Date calculation (do this first):**
1. Get today's date in Australia/Sydney timezone.
2. Subtract one day — that is the **target date** for this run.
3. The target filename is: `outputs/daily-finance/YYYY-MM-DD-financial-news-report.md` (using the target date).
4. Check whether that file already exists in this repository.
   - If it does **not** exist: generate and save the report for that date, then send the Discord message.
   - If it **already exists**: do not generate a new report. Send the Discord message using the existing report for that date.

Do **not** re-use or re-send any report from a date other than the target date.

## Report rules

- Use the template in `templates/daily-financial-report-template.md`.
- Keep the report around one page.
- Include source links where available.
- Do not invent market data.
- Clearly state when data is unavailable.
- Do not provide personalised financial advice.
- Save only the final report file.

## Suggested public sources

Use reliable public sources to research the target date's market activity:
- Reuters business and markets coverage
- Yahoo Finance / market data pages
- MarketWatch
- CNBC Markets
- Federal Reserve official releases
- Reserve Bank of Australia releases
- ASX market information
- Commodity market pages for gold, oil, and iron ore

Prioritise market-moving information over general news.

## Discord delivery

After saving (or confirming) the report, send the **full report content** as a Discord message using the environment variables:

- DISCORD_BOT_TOKEN
- DISCORD_CHANNEL_ID

Discord messages have a 2000-character limit. Send the report in multiple sequential messages if needed — split at logical section boundaries (e.g. after the Executive Summary, after the Market Snapshot, etc.). Do not use file attachments.

Do not include any Discord token or secret in this repository.
