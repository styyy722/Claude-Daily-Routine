# Claude Daily Finance Routine Instructions

This repository stores automated daily financial news reports generated by Claude Routine.

## Main task

Each routine run should generate one markdown report for the previous calendar day using Australia/Sydney time.

Save the report to:

outputs/daily-finance/YYYY-MM-DD-financial-news-report.md

## Report rules

- Use the template in `templates/daily-financial-report-template.md`.
- Keep the report around one page.
- Include source links where available.
- Do not invent market data.
- Clearly state when data is unavailable.
- Do not provide personalised financial advice.
- Do not overwrite old reports unless correcting an error for the same date.
- Save only the final report file.

## Discord delivery

After saving the report, send a short Discord message using the environment variables:

- DISCORD_BOT_TOKEN
- DISCORD_CHANNEL_ID

The Discord message should include:

- Report title
- Report date
- 3-bullet executive summary
- GitHub file path or link to the full report

Do not include any Discord token or secret in this repository.
