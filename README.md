# Claude Daily Finance Routine

This repository stores daily one-page financial news reports generated by a Claude Routine.

The routine is designed to run every morning, research the previous calendar day’s financial market news, and save a concise markdown report into the `outputs/daily-finance/` folder.

## Purpose

The goal of this project is to create a simple automated reporting system for daily financial news.

Each report should summarise the most important market-moving developments from the previous day, including equities, interest rates, FX, commodities, and major company or sector news.

## How It Works

```text
Claude Routine
    ↓
Runs every morning
    ↓
Researches previous day’s financial news
    ↓
Generates a one-page markdown report
    ↓
Saves the report into GitHub
